Navigation

Shortcut Description
t + f Open search
/ + <content-letter> Toggle search specifiers
t + <panel id> Toggles panel visibility [no clear id's yet]
g + g Open content object in current window
j/down next focusable object
k/up next focusable object
q/Esc Focus anscestor
l/Enter Focus first descendant

Content Creation

Shortcut Description
c + <content-letter> Insert child to focused object.
v + <content-letter> Insert undirected to focused object.
ctrl + alt + shift + d Full deletion of Content Object.

Project

Shortcut Description
Shift + 0 Load focused project as current.
Alt + 0 Create new project and load as current.

States

Shortcut Description
Shift + 1/2/3 Set focused object as state 1/2/3.
1/2/3 Set focused object as child of object in state 1/2/3.
4/5/6 Set focused object as undirected of object in state 1/2/3.
7/8/9 Set focused object as parent of object in state 1/2/3.

Object context menus

Shortcut Description

p Content properties.
m toggle menu focus
